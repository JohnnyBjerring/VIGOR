@startuml
title SSD - UC02 Se borgere på afdeling

actor Personale
participant "VIGOR System" as System

== Main Success Scenario ==

Personale -> System : requestCitizenOverview()
System --> Personale : showCitizenList(citizens)

== Alternative Flow - Ingen borgere på afdeling ==

Personale -> System : requestCitizenOverview()
System --> Personale : showEmptyCitizenList(message)

== Alternative Flow - Manglende adgang ==

Personale -> System : requestCitizenOverview()
System --> Personale : showAccessDenied(message)

== Alternative Flow - Systemfejl ==

Personale -> System : requestCitizenOverview()
System --> Personale : showError(errorMessage)
@enduml
