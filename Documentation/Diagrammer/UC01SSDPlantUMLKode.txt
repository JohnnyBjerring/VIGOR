title SSD - UC01 Log ind (rollebaseret)

actor Personale
participant "VIGOR" as System

== Main Success Scenario ==

Personale -> System : enterCredentials(email, password)
System --> Personale : loginAccepted()
System --> Personale : navigateToStartPage(role)

== Alternative Flow - Ugyldige credentials ==

Personale -> System : enterCredentials(email, password)
System --> Personale : loginRejected(errorMessage)

== Alternative Flow - Manglende rolle ==

Personale -> System : enterCredentials(email, password)
System --> Personale : accessDenied(message)

@enduml
