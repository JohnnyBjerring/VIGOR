# UI fix – login-knap

## Ændring

Login-knappen er gjort tydeligere med:

- mørkere VIGOR/navy gradient
- stærkere tekstkontrast
- tydeligere border og skygge
- bedre hover/focus/disabled states
- højere CSS-specificitet, så global button-styling ikke vasker knappen ud

## Berørte filer

- `VIGOR.MAUI/Components/Pages/Login.razor.css`

## Formål

Knappen var for lys/utydelig på login-siden. Rettelsen gør primær handling tydeligere og mere professionel uden at ændre login-flowet.
